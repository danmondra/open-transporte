# Open Transporte
The unificative solution of the transportation system in Mexico.

It uses ['Open Payments From Interledger']('https://interledger.org/open-payments') to financially interoperate through the transportation system.

## Tech Stack
- TypeScript
- NodeJS
- ExpressJS
- NextJS
- Docker
- PostreSQL
